myFlix Project

Create a REST API with Node.js

Objective:

To build the server-side component of a "movies" web application.

The web application will provide users with access to information about different movies, directors, and genres.

Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL Endpoint HTTP Method Request Body Request Example Response Body
Get a list of all movies /movies GET None /movies JSON
Get information about one specific movie by title (use %20 for spaces) /movies/:title GET None /movies/Ace%20Ventura JSON
Return data about genre(description) by name/title (use %20 for spaces) /movies/genres/:genre GET None /movies/genres/comedy JSON
Return data about a director by name (use %20 for spaces) /movies/directors/:name GET None /movies/directors/Christopher%20Nolan JSON
